Installer Windows 11 sur un PC non-supporté

Tutoriel vidéo

Activer Windows 10 et 11 gratuitement

Tutoriel vidéo

Tutoriel écrit

Tout d'abord, téléchargez le script en cliquant ici.

Cherchez le fichier dans votre ordinateur (il se trouve dans votre dossier de téléchargements par défaut).

Effectuez un clic-droit sur le fichier et sélectionnez "Exécuter en tant qu'administrateur" (vous devez disposer des droits administrateurs au préalable). Dans le script, appuyez sur "1".

Une fois le script éxecuté, Windows devrait être activé. Pour vérifier cela : rendez vous dans les Paramètres Windows > Système > Informations système.

S'il est écrit "Windows est activé" alors cela signifie que le processus d'activation a fonctionné.
